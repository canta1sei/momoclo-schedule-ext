# ももクロ スケジュール改善 Chrome拡張機能

ももいろクローバーZ 公式スケジュールページ（[momoclo.net/schedule](https://www.momoclo.net/schedule)）を、より見やすい UI に丸ごと置き換える Chrome 拡張機能です。

![スクリーンショット](docs/screenshot.png)

---

## 解決している問題

| 問題 | 解決策 |
|------|--------|
| カテゴリタブを切り替えるたびにページ全体が再読み込みされる | クライアントサイドフィルタリングで再 fetch 不要 |
| 「今日・今週に何があるか」確認するのに View More を何度も押す必要がある | 今月〜3ヶ月先を起動時に一括取得して全表示 |
| 同じ日のイベントが別々の枠で並んでいて把握しにくい | 同日イベントを1ブロックにグルーピング |

---

## 機能

- **複数月一括取得** — 今月〜3ヶ月先を `Promise.all` で並列 fetch
- **日付グルーピング** — 同じ日のイベントをまとめて表示
- **TODAY / 今週 バッジ** — 今日・今週のイベントを視覚的にハイライト
- **カテゴリフィルター** — ALL / LIVE / EVENT / TV / RADIO など、再 fetch なしで絞り込み
- **過去イベント非表示** — 今日以降のスケジュールのみ表示

## インストール

Chrome Web Store からのインストールは未対応のため、手動でロードしてください。

1. このリポジトリを ZIP でダウンロード、または `git clone`
2. Chrome で `chrome://extensions` を開く
3. 右上の **デベロッパーモード** をオン
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. ダウンロードしたフォルダを選択

`https://www.momoclo.net/schedule` を開くと自動的に改善UIに切り替わります。

## ファイル構成

```
momoclo-schedule-ext/
├── manifest.json   # Manifest V3 設定
├── content.js      # メインロジック（fetch・パース・レンダリング）
└── style.css       # ももいろテーマのスタイル
```

外部ライブラリ・ビルドツール不要。バニラ JS のみで動作します。
